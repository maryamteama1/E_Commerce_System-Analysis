# E-Commerce System Analysis & Performance Dashboard

## 📌 Project Overview
This repository contains a comprehensive **E-Commerce Performance Dashboard** designed to analyze and track business performance, sales trends, and customer behavior. The dashboard provides actionable insights into key revenue drivers, order fulfillment statuses, and product category distributions over multiple years (2024–2026).

## 🔍 Features & Detailed Analysis

### 1. Advanced Interactive Filtering
*   **Yearly Slicers:** Filter data dynamically across **2024, 2025, and 2026**.
*   **Order Status Slicers:** Segment metrics by operational status: `Cancelled`, `Delivered`, `Pending`, `Processing`, and `Shipped`.

### 2. Customer Insights (`Top Customers`)
*   A horizontal bar chart identifying high-value customers based on the total count of orders.
*   **Top performing customer:** *Michael* leads with the highest order frequency, closely followed by *David* and *James*.

### 3. Sales Trends (`Monthly Sales Trend`)
*   A line chart demonstrating seasonality and revenue fluctuations from January to December.
*   Helps business stakeholders pinpoint peak sales months (e.g., April, August, October) versus slower periods (e.g., February, September).

### 4. Product Performance (`Sales By Category`)
*   A detailed pie chart illustrating sales distribution and percentages across major inventory categories:
    *   **Sports & Outdoors** (Leading category with 287 sales / 23.3%)
    *   **Automotive** (250 sales / 20.29%)
    *   **Toys & Games** (243 sales / 19.72%)
    *   **Home Appliances** (230 sales / 18.6%)
    *   **Electronics** (222 sales / 18.02%)
